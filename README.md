# plusコマンド

![test](https://github.com/reniker1/robosys203r/actions/workflows/test.yml/badge.svg)

標準入力から読み込んだ数字を足す。

## 実行例

* seq 5 | ./plus と入力すると15を出力

## 必要なソフトウェア

* Pythpon
  * テスト済みバージョン: 3.7~3.10

## テスト環境

* Ubuntu
